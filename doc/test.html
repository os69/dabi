Introduction

dobi (dombinding) is a javascript data binding library for building HTML5 user interfaces. Data binding connects javascript artifacts (objects, properties of objects, lists) with their UI counterparts. For instance a javascript list a=['Hello','World!'] may be represtend in the DOM by a corresponding list 
<ul>
    <li>Hello</li>
    <li>World!</li>
</ul>
Data binding does mean that when the javascript list is changed (for instance a.push('Hi!')) then the UI list reflects the change. Databinding can be bidirectional: You can bind an input field in the UI to an object property. Entering values into the input field will change the object property. Changing the object property (programmaticaly) will change the value shown in the input field.

Installation

Download the archive or just the minified javascript library dobi.min.js. Put the sources in the WWW folder of your webserver.

Using

Minified Javascript

The fastest way of using dobi is to include the minified javascript library into your HTML document (Example): 
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="utf-8">
    <script src="../rel/dobi.min.js"></script>
    <title>Hello World</title>
</head>

<body>
...
</body>

</html>

Uncompressed Javascript

Instead of the minified library you can also use the uncompressed source files. This is useful for debugging or if you want to change dobi (Example): 
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="utf-8">
    <script src="../dobi/eventing.js"></script>
    <script src="../dobi/list.js"></script>
    <script src="../dobi/property.js"></script>
    <script src="../dobi/binding.js"></script>
    <title>Hello World</title>
</head>

<body>
...
</body>

</html>

RequireJS

You can load the dobi library by RequireJS (Example): 
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="utf-8">
    <script src="../tp/require.js"></script>
    <title>Hello World</title>
</head>

<body>


    <div id="output"></div>


    <script>
        requirejs.config({
            baseUrl: '..',
        });

        require(['dobi/binding'], function (bindingModule) {
        ...            
        });
    </script>
</body>

</html>

Eventing

dobi is based on a eventing library which connects dom nodes with javascript objects. When not using RequireJS you the eventing library is loaded into the global namespace dobi.eventing.

Raising Events

A sender can raise an event by calling eventing.raiseEvent: 
var sender = {

    setValue: function (value) {
        this.value = value;
        eventing.raiseEvent(this, 'ValueChangedSignal', value);
    }

};

Subscribe to Events

Subscribe to events by eventing.subscribe: 
// =======================================================================
// receiver
// =======================================================================
var receiver = {

    handler1: function (event) {
        console.log('handler1: received event', event.message);
    },

    handler2: function (event) {
        console.log('handler2: received event', event.message);
    }

};

// =======================================================================
// subscription to events
// =======================================================================
eventing.subscribe(sender, 'ValueChangedSignal', receiver, receiver.handler1);
eventing.subscribe(sender, 'ValueChangedSignal', receiver, 'handler2');
eventing.subscribe(sender, 'ValueChangedSignal', receiver, function(event){
    console.log('handler3: received event', event.message); // this=receiver
});
