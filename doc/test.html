Introduction

dobi (dombinding) is a javascript data binding library for building HTML5 user interfaces. Data binding connects javascript artifacts (objects, properties of objects, lists) with their UI counterparts. For instance a javascript list ['Hello','World!']may be represtend in the DOM by a corresponding list 
<ul>
    <li>Hello</li>
    <li>World!</li>
</ul>
Data binding does mean that when the javascript list is changed then the UI list reflects the change. Databinding can be bidirectional: In case you have an input field in the UI entering values into the field can change an javascript object property.

Installation

Download the archive or just the minified javascript library dobi.min.js. Put the sources in the WWW folder of your webserver.

Using

Minified Javascript

Include the minfied javascript library into your HTML document: 
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="utf-8">
    <script src="../rel/dobi.min.js"></script>
    <title>Hello World</title>
</head>

<body>

    <div id="templates">
        <div data-bind="data/data1"></div>
    </div>

    <script>
        var data = {
            data1: '4711'
        };
        dobi.binding.run(window, document.getElementById('templates'));
    </script>
</body>

</html>

RequireJS

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="utf-8">
    <script src="../tp/require.js"></script>
    <title>Hello World</title>
</head>

<body>


    <div id="output"></div>


    <script>
        requirejs.config({
            baseUrl: '..',
        });

        require(['dobi/binding'], function (bindingModule) {

            var data = {
                data1: '4711'
            };

            bindingModule.bindString(data.data1, document.getElementById('output'));
            
        });
    </script>
</body>

</html>
